To add media in HTML, we use special tags for images, audio, and video. An image is added using the img tag, where we give the file path using the src attribute. To add audio, we use the audio tag, and to add video, we use the video tag. Inside audio and video tags, we use the source tag to specify the media file. The controls attribute is used so that play, pause, and volume buttons appear. These tags help make a webpage more interactive and attractive by including media content like pictures, music, and videos.
